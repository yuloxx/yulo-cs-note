## üõ†Ô∏è Build with MkDocs

This project uses **MkDocs** to generate the documentation website.

### 1. Install Python

Ensure Python 3.x is installed:

```bash
python --version
```

### 2. Install MkDocs

Install MkDocs via pip:

```bash
pip install mkdocs
```

### 3. Navigate to the Project Directory

Change to the folder containing `mkdocs.yml`:

```bash
cd /path/to/your/project
```

### 4. Start the Local Server

Run:

```bash
mkdocs serve
```

### 5. Open the Documentation

Open the local URL displayed in the terminal, usually:

```
http://127.0.0.1:8000/
```

Your documentation site will now be available locally.
